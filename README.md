# Board Games Companion Apps

Коллекция **неофициальных компаньон-приложений** и модов для настольных игр.

Проекты предназначены для **упрощения игрового процесса** и доступны в виде:
- **WEB-версий**
- **Android-приложений**
- **модов для Tabletop Simulator**

Приложения используют PnP-изображения, свободно доступные в интернете.  
Часть изображений и звуков создана или обработана с помощью нейросетей.

> ⚠️ Все материалы **не являются официальными** и **не заменяют оригинальные настольные игры**.
